# XAI Heart Disease Project

This repository contains the code for the Explainable AI assignment.

The project compares three explanation methods:
- SHAP
- LIME
- ELI5

The methods are evaluated using three metrics:
- Faithfulness
- Stability
- Completeness

Dataset: Heart Disease Dataset (UCI / Kaggle)

Author: Leonie Leeuwis
Radboud University
